## Recursos:
- **Navegação por setas**: Usuários podem navegar entre as imagens
- **Indicadores visuais**: Pontos na parte inferior mostram qual imagem está ativa
- **Clique nos indicadores**: Usuários podem clicar diretamente em qualquer imagem

## Formatos aceitos:
- .jpg
- .jpeg
- .png
- .webp

## Tamanho recomendado:
- Largura: 600px
- Altura: 400px
- Proporção: 3:2

Se as imagens não estiverem disponíveis, o sistema usará imagens de exemplo automaticamente.
